🔍 各ツール最新アップデート＆ユーザーの声

🚀

Antigravity / Gemini Code Assist

Google DeepMind — 第6回アップデート
🆕 Code Assist 個人無料化 VS Code / JetBrains対応 CLI有料化継続

📢 最新アップデート

  • Gemini Code Assist 個人完全無料化 🔥 — 3月から個人開発者向けにVS Code・JetBrains用プラグインを制限なし・無料で提供開始。従来の有料サービスが無料に。企業・チーム向けとの棲み分け戦略
  • Gemini 3.1 Pro API — $2/M input・$12/M outputで提供開始。Gemini 3.1 Flash-LiteはGemini 2.0 Flashより30%安く、低コストAPIとして注目
  • Desktop Intelligence（Mac）テスト継続 — 画面コンテンツとの深い統合機能。プライベートテスト段階
  • Claude API価格（参考） — Sonnet $3/M input・$15/M output。HaikuはGemini Flash Liteと同価格帯で競合

💰 構成の変化

  • Gemini Code Assist（IDE Plugin）: 個人は完全無料 ✅
  • Antigravity CLI（Gemini CLI）: 無料はFlashモデルのみ（有料化継続）
  • Google AI Pro ($19.99/月): CLI Proモデル＋Workspace統合
  • Google AI Ultra ($49.99/月): 最大クォータ

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ Code Assist無料化の真意を疑う声 — 「無料にして囲い込んでから有料化する戦略では？」との懐疑論も。Cursor/Copilotに比べてエコシステムが弱い
  • ⚠️ MCP未対応（継続） — Cursor/Copilot/Claude Codeに差をつけられたまま。MCP標準化が進む中で不利
  • ⚠️ CLI有料化への実害が顕在化 — 3/25以降の無料Flash限定で「実務では使いにくい」との声が増加
  • ✅ Code Assist無料化は歓迎 — 「IDEで軽く使う分には最強の無料ツールになった」との評価も

🔑 第6回の所感

Gemini Code AssistのIDE Plugin無料化は「CLI有料化の代償措置」とも読める。 ターミナル派にはコスト増、IDE派には朗報という二極化。 当サイトはAntigravity（CLI）で開発を継続しているが、Code Assist無料化はIDEユーザーへの強力な選択肢となった。

🤖

Claude Code

Anthropic — 第6回アップデート
🔥 採用率No.1 $17/月〜新プラン ❤️ 開発者満足度最高

📢 最新アップデート

  • 開発者採用率No.1（2026年） 🔥 — 調査サイト複数で「2026年最も採用されているAIコーディングツール」トップに。リファクタリングと自律タスクで圧倒的評価
  • 新プラン $17/月〜 — Claude Code Pro（$17/月）が新設され、より入りやすい価格帯に。用途別に$17/$100を選択可能
  • CLAUDE.mdによるプロジェクト記憶 — ルールファイルによるコンテキスト管理が普及。Hookやスキル機能で「覚えているAI」を実現
  • Claudeがff22件の脆弱性を発見（Mozilla共同） — セキュリティ分野でも存在感。Claude Code + セキュリティリサーチの事例が増加
  • Spring Break特典継続（〜3/28） — オフピーク・週末は使用量2倍キャンペーン継続中

💰 価格体系（新プラン追加）

  • Claude Code Pro $17/月: Sonnet利用可・基本タスク向け（新設）
  • Claude Code Max $100/月: 5〜20倍クォータ・Opus利用可
  • API直接利用: Sonnet $3/M input・$15/M output
  • Team $125/人/月: プレミアムシート（年間契約）

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ シークレット漏洩リスク（業界課題に拡大） — Claude Code共同コミットでのシークレット漏洩率2倍問題は継続。ただしCursor/Copilotも同様の問題が報告され、業界全体の課題に
  • ⚠️ コンテキストウィンドウ管理の難しさ — 長期セッションでコンテキストが溢れ「以前の指示を忘れる」問題。CLAUDE.mdでの事前設定が推奨される
  • ⚠️ Pentagon問題（継続） — Anthropicの軍事利用制限方針とペンタゴンの「リスク指定」の対立は未解決
  • ✅ auto-plan + voice mode好評（継続） — ワークフロー革新として引き続き高評価

🔑 第6回の所感

採用率No.1は実測パフォーマンスが証明した結果。 リファクタリング71%短縮、DevOps設定90%成功率は他ツールを一線画く。 $17/月新プランで入門ハードルも下がり、2026年後半もトップを維持する公算が高い。

Cursor

Anysphere — 第6回アップデート
無料プランあり 🏆 新規実装最速 1M+ DAU継続

📢 最新アップデート

  • 実測データで新規実装最速を証明 🏆 — CRUD実装17分（手動65分）は5ツール中トップ。Tab補完の精度と低遅延がCRUD等の定型実装で圧倒的な結果
  • $50B評価額交渉の継続報道 — AI開発ツール企業として異例の評価額。1M+ DAUの実績が支えている
  • 「AITool for VSCode」対比での優位性 — VS Codeと互換性を保ちつつ、AIファーストのUXで差別化。「乗り換えて正解」の声が多い
  • Marketplace拡充継続 — Atlassian、HuggingFace等30+パートナープラグイン。カスタムワークフロー構築が容易

💰 価格体系（変更なし）

  • Hobby（無料）: 限定リクエスト + 7日間Pro体験
  • Pro $20/月: 無制限Tab補完 + $20クレジットプール
  • Pro+ $60/月: 3倍クレジット
  • Ultra $200/月: 20倍使用量 + 優先アクセス

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ 実質コストの高さ（最大の不満、継続） — ヘビーユーザーで$60-80/月。Composer 2のクレジット消費でさらに上がる可能性。$20/月で済む人は少数
  • ⚠️ リファクタリングではClaude Codeに負ける — 実測データで明確に。CursorはCRUD向き、複雑なリファクタはClaude
  • ⚠️ Composer 2ロックイン懸念（継続） — 自社モデル依存でBYOKの自由度が低い
  • ✅ 新規実装スピードは圧倒的 — 「プロトタイプを爆速で作る」用途では依然最強との評価

🔑 第6回の所感

「新規実装はCursor、リファクタリングはClaude Code」という使い分けが実測データで確定した。 一つのツールで全てをカバーするより、タスクに応じた使い分けが2026年のベストプラクティス。 $50B評価額が示す市場の期待は高いが、コスト問題が普及の壁になっている。

GitHub Copilot + Codex

GitHub / OpenAI — 第6回アップデート
無料プランあり 🆕 Codex復活 シェア42% / 最安$10/月

📢 最新アップデート（Codex復活が最大トピック）

  • Codex復活（開発者ツール市場再参入） 🔥 — OpenAIがCodexをリポジトリ規模対応・CI連携強化で再公開。マルチファイルプロジェクト構造を理解し、automated code review/generationをCIに組み込める。2026年の新たな競争軸
  • GitHub Copilot CLI 正式リリース（3/5） — ターミナル向けコーディングエージェントが正式版に。Claude Code同様のCLIワークフローをCopilotで実現
  • HubSpot Sidekick 事例 — マルチモデルAIコードレビューで90%高速化・エンジニア承認率80%という大規模導入事例が公開。CopilotのエンタープライズROIが可視化
  • OpenAI vs Microsoft緊張の余波 — OpenAIがGitHub対抗の独自コードリポジトリを開発中との報道。Copilotの親会社関係が変わる可能性も

💰 価格体系（変更なし）

  • Free: 月2,000補完 + 50プレミアムリクエスト
  • Pro $10/月: 無制限補完 + 300プレミアムリクエスト
  • Pro+ $39/月: 1,500リクエスト + 優先アクセス
  • Business $19/人/月: 組織管理
  • Enterprise $39/人/月: （GH Enterprise Cloud必須 → 実質$60）

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ Codex復活への懐疑 — 「以前のCodexは精度が低かった。今回は本当に良くなったのか」との検証待ちの声
  • ⚠️ 愛され率9%（継続） — シェア42%に対してユーザー満足度は低いまま。「使わざるを得ない企業ツール」のポジション
  • ⚠️ OpenAI-Microsoft対立の不確実性 — 親会社関係の不安定さがエンタープライズ採用判断に影響する可能性
  • ✅ $10/月の安さは競合なし — 5ツール中最安。Copilot CLIの正式リリースで機能面も改善

🔑 第6回の所感

Codex復活とCLI正式リリースは「Copilotの本気」を示す。 HubSpotの大規模導入事例はエンタープライズ向けROIを証明。 ただしOpenAI-Microsoft対立という根本的不確実性が、長期投資先としての評価を難しくしている。

Windsurf

Cognition (元Codeium) — 第6回アップデート
無料プランあり 🆕 Arena Mode $15/月 最安水準

📢 最新アップデート

  • Arena Mode（複数モデル対決機能） 🆕 — 複数のAIモデルを同一タスクで同時実行し、出力を比較できる新機能。「どのモデルが自分のコードに最適か」を実測で判断できる差別化機能
  • Cascade機能の継続改善 — マルチファイル編集チェーン機能。シンプルなリファクタリングでの高速性は引き続き評価が高い
  • 350+エンタープライズ顧客を維持 — 既存顧客は安定しているが、新規獲得の勢いはCursorに劣る

💰 価格体系（変更なし）

  • Windsurf Free: 基本機能（制限付き）
  • Windsurf Pro $15/月: フル機能 — 5ツール中2番目に安い
  • Devin: エンタープライズ向け（要問い合わせ）

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ 将来の不透明さ（継続・最大の課題） — Cursor・Claude Codeとの差異が縮まり、Arena Modeで勝負するしかない状況
  • ⚠️ リファクタリングで最下位 — 実測データでリファクタリング85分は3位（Cursor55分、Claude 47分より遅い）
  • ⚠️ 機能の「速い判断」が誤りになるリスク — 速度重視のあまり意図しない変更を生む「思い込みコーディング」問題が継続
  • ✅ Arena Modeは差別化として有望 — 「どのモデルを使うか自分で選べる」哲学は他ツールにない強み

🔑 第6回の所感

Arena Modeは「エコシステムへの囲い込みに抵抗する姿勢」として評価できる。 しかし実測データでは最速でも最高精度でもなく、差別化の軸が弱い。 $15/月の安さとエンタープライズ実績が頼みの綱。長期投資には慎重な判断が必要。
