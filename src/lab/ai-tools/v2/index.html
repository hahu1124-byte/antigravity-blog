Claude Code

Anthropic — 第2回アップデート
Pro $20/月 Code Review 新機能 🔥 急成長

📢 最新アップデート

  • Code Review（3月9日発表） — マルチエージェントAIがPRをスキャンし、ロジックエラーを検出・修正提案。レビューコメントの質が16%→54%に向上（社内テスト）
  • Claude Cowork（1月リリース） — 非コーダー向けGUI版。ビジネスプロフェッショナル向け「AIエンプロイヤー」
  • Claude Opus 4.6（2月5日） — 100万トークンコンテキスト、14.5時間のタスク完了ホライゾン
  • Claude Sonnet 4.6（2月17日） — Opusに近い性能でコスト効率が高い
  • Claude Code Security — オープンソースプロジェクトで数百の未検出脆弱性を発見
  • リモートコントロール — スマートフォンからQRコードで接続・操作可能に
  • 新スラッシュコマンド — `/simplify`（コード簡素化）、`/batch`（バッチ操作）追加

💰 価格体系（更新）

  • Pro $20/月: Sonnet/Opus利用可、週40-80時間のClaude Code利用
  • Max $100/月: Pro の5〜20倍のクォータ + 早期アクセス
  • Team $125/人/月（Premium seat、年間契約）

Windsurf

Codeium → Cognition買収 — 第2回アップデート
無料プランあり Pro $15/月 ⚠️ 買収による変化

📢 最新アップデート

  • Cognitionが買収 — OpenAIの$30億買収は不成立。代わりにCognition（Devin開発元）が買収、Googleも一部キースタッフを採用
  • Supercomplete — 次の行だけでなく「次の3ステップ」を予測する新しいオートコンプリート
  • Code Integrity Layer — AI生成コードの脆弱性を自動スキャン、セキュリティ基準の遵守を確保
  • 新モデル追加 — Gemini 3.1 Pro（2月19日）、Claude Sonnet 4.6（2月17日）、GLM-5、Minimax M2.5
  • Context API（ベータ） — Codeiumの推論エンジンを社内ツールに統合可能に

⚠️ 買収後の展望

CognitionがDevinの技術とWindsurfを統合する方向性が示唆されており、今後の大幅な変化が予想されます。Windsurf単体としてのロードマップは不透明になる可能性があります。
