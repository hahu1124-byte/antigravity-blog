🚀

Antigravity

Google DeepMind — 第4回アップデート
無料プレビュー継続 Workspace AI 🆕 🔥 エコシステム拡大

📢 最新アップデート

  • Gemini × Workspace統合（3/10） — Docs「Help me create」でドライブ/Gmail/Chatの情報を合成してドラフト自動生成。Sheets AIはスプレッドシート全体を構築・編集（SpreadsheetBenchで70.48%達成）。Slides AIでプレゼン自動生成。「Ask Gemini in Drive」でファイル横断セマンティック検索
  • Gemini Embedding 2（3/10） — テキスト・画像・動画・音声・PDFを統一ベクトル空間で処理するマルチモーダル埋め込みモデル。RAGアプリケーションの精度が大幅向上
  • Pixel エージェント機能 — サードパーティアプリ内でAIがタスク実行する「エージェント機能」をスマホに導入。モバイル開発ワークフローにも影響
  • Nano Banana 2 更新 — 即時高精度画像生成がGemini 3.1 Fast/Thinking/Proモードのデフォルトに
  • エージェントファースト設計は変わらず — エディタ・エージェントマネージャ・ブラウザが一体化した統合IDE

💰 価格体系（変更なし）

  • Individual（無料）: 寛大なレート制限付きで全モデル利用可
  • Developer: Google AI Pro ($19.99/月) / Ultra ($49.99/月) で制限緩和
  • Team / Organization: Workspace / Google Cloud経由

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ Pro版 週間制限問題（継続） — 第3回で報告した「週間キャップ」問題は未解決。数日間ロックアウトされるユーザーの報告は依然として多い
  • ⚠️ Geminiアプリに広告導入の報道 — 無料プランでのAI体験に広告が挟まる可能性。ユーザーから「無料は広告付き、広告なしは有料」の二極化を懸念する声
  • ✅ Workspace統合は好評 — 「Sheetsで丸ごとスプレッドシート構築はすごい」「Docs連携でドキュメント作成が劇的に早くなった」との声
  • ⚠️ MCP未サポート（継続） — Model Context Protocol未対応。外部データソース連携に制約
  • 3月のアカウント凍結問題は沈静化 — 大量凍結の報告は減少傾向

🔑 第4回の所感

Workspace統合でGoogleのエコシステム力が存分に発揮された。 無料枠の寛大さは5ツール中No.1。一方でPro版週間制限は未解決で、 さらに広告導入報道が加わり「無料の品質」に微妙な影が。 当サイトは引き続きAntigravityで開発中だが、制限時はClaude Codeに切り替える運用。

🤖

Claude Code

Anthropic — 第4回アップデート
Pro $20/月 🔥 コスト革命 ❤️ 愛され率46%

📢 最新アップデート

  • 長文サーチャージ撤廃（3/16）🔥 — Opus 4.6 / Sonnet 4.6の100万トークンコンテキストが追加料金なしに。大規模コードベース一括解析のコスト障壁が消滅。GPT-5.4との100万トークン競争で「Claude＝高い」のイメージを払拭
  • ダブル使用量キャンペーン（3/13〜27） — 全プラン（Free/Pro/Max/Team）でオフピーク＆週末は使用量2倍。Web/Desktop/Mobile/Code/Excel/PPTすべて対象。#QuitGPTからの乗り換えユーザー定着を狙う
  • チャート・インライン可視化β（3/12） — チャット内でインタラクティブなグラフ・図表を直接作成可能に。データ分析の民主化が加速
  • メモリ機能全ユーザー開放（3/2〜） — 無料ユーザーでもチャット履歴からのメモリ利用可能。他AIからの記憶移行ツールも提供
  • Claude Code年間売上25億ドル — 2025年1月ローンチから1年強で$25B規模。開発者の「一軍ツール」に完全昇格

💰 価格体系（長文サーチャージ撤廃で実質値下げ）

  • Pro $20/月: Sonnet/Opus利用可 — 長文サーチャージなしで100万トークン
  • Max $100/月: 5〜20倍のクォータ
  • Team $125/人/月（Premium seat、年間契約）
  • API利用: ヘビーユーザーは$100〜$300/月程度（サーチャージ撤廃で値下がり）

😤 ユーザーの不満・課題（継続フォロー）

  • ✅ 障害頻度は改善傾向 — 第3回で報告した3月の連発障害（3/2, 3/3, 3/11）以降、大規模障害の報告は減少。ただしピーク時の速度低下は依然あり
  • ⚠️ Proプラン レート制限（継続） — ヘビーユーザーが作業中断される問題は未解決。Max ($100/月) への移行が推奨される状況
  • ⚠️ Code Review高額（継続） — PRあたり平均$15〜$25は変わらず。「AIが生成→AIが高額レビュー」の皮肉な構図
  • ⚠️ 中国AI蒸留問題 — 3社による24,000偽アカウント・1,600万回のやりとりが発覚。知的財産防衛の新課題
  • ✅ サーチャージ撤廃は好評 — 「100万トークン追加料金なしは神」「これでClaudeがLLMの主力に確定」との声が圧倒的

🔑 第4回の所感

長文サーチャージ撤廃＋ダブル使用量CPは今回の最大のニュース。 第3回で指摘した「障害連発」問題も改善傾向にあり、「愛され率46%」の地位はさらに盤石に。 ただしProプランのレート制限はMax ($100/月) への誘導の側面もあり、 ヘビーユーザーの「実質コスト」はCursorに迫る可能性がある。

Cursor

Anysphere — 第4回アップデート
無料プランあり Automations 安定化 ARR $500M+

📢 最新アップデート

  • Automations安定化 — 第3回で紹介した常時オンエージェントが安定化フェーズに。Slack/GitHub/PagerDutyトリガーの信頼性が向上
  • 統合ブラウザ改善 — Chrome内蔵のIDEブラウザがDevTools連携をさらに強化。フロントエンド開発のワークフローがさらにシームレスに
  • プラグインマーケットプレイス拡大 — プリビルトソリューションの数が増加。カスタムワークフロー構築が容易に
  • Tab補完での生産性向上 — 30-40%の生産性向上を報告する開発者が引き続き増加

💰 価格体系（変更なし）

  • Hobby（無料）: 限定リクエスト + 7日間Pro体験
  • Pro $20/月: 無制限Tab補完 + $20クレジットプール
  • Pro+ $60/月: 3倍のクレジットプール
  • Ultra $200/月: 20倍の使用量 + 新機能優先アクセス
  • Teams $40/人/月: 共有チャット・ルール・SAML SSO

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ コスト問題は継続（最大の不満） — 実質$60-80/月に達するヘビーユーザー多数。Claude長文サーチャージ撤廃でコスト対比がさらに不利に
  • ⚠️ Automationsのコスト — クラウドエージェントクレジット消費。頻繁なAutomationsで割り当てを使い果たすリスク
  • ⚠️ CVE脆弱性（継続監視） — MCPoison/CurXecute（CVE-2025-54135/54136）の修正は適用されたが、セキュリティ意識の高いユーザーは引き続き警戒
  • ⚠️ メモリ使用量 — 大規模コードベースのインデックスでRAM枯渇→レンダラークラッシュの報告は継続
  • ✅ Automationsの好評 — 「ゲームチェンジャー」との評価は変わらず。24時間稼働のAIエージェントは唯一無二の価値

🔑 第4回の所感

Automationsの革新性は依然No.1。しかしClaudeの値下げにより 「Cursorのコストパフォーマンス」が相対的に悪化。$20/月の「額面」では Claude Proと同じだが、実質コストではCursorが大幅に上回る。 Automationsの価値を認めるユーザーには最適だが、コスト重視なら他ツールに分がある。

GitHub Copilot

GitHub / OpenAI — 第4回アップデート
無料プランあり Copilot Cowork 🆕 シェア42% / 愛され9%

📢 最新アップデート

  • Microsoft Copilot Wave 3 — 指示を自律的に遂行する「Copilot Cowork」機能を導入。エージェント型の自律タスク実行で、Cursor Automationsに対抗
  • Copilot for Small Business — 中小企業向けの新プランを発表。エンタープライズ以外の市場にも手を広げる
  • Codex Security（3/6） — OpenAI発のAI搭載コード監査ツールがプレビュー版で公開。AI生成コードの脆弱性対策として注目
  • ChatGPT for Excel — GPT-5.4搭載でスプレッドシート内から直接AI活用。Gemini×Sheets対抗
  • GPT-5.4安定稼働 — 第3回で統合されたGPT-5.4のPC操作機能が安定フェーズに。Jira連携も安定化

💰 価格体系（変更なし）

  • Free: 月2,000補完 + 50プレミアムリクエスト
  • Pro $10/月: 無制限補完 + 300プレミアムリクエスト
  • Pro+ $39/月: 1,500リクエスト + 優先アクセス
  • Business $19/人/月: 組織管理 + ポリシー制御
  • Enterprise $39/人/月: （⚠️ GH Enterprise Cloud $21必須 → 実質$60/人/月）

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ 愛され率9%（継続） — 市場シェア42%にもかかわらず「most loved」は最低水準。「使われているが好かれていない」の構図は変わらず
  • ⚠️ 超過課金への混乱（継続） — 300リクエスト超過後の$0.04/件課金に混乱するユーザーは依然多い
  • ✅ Copilot Coworkへの期待 — Wave 3発表でエージェント機能追加。「やっとCursor/Claudeに追いついた」との評価
  • ⚠️ コード品質（継続） — 表面的なレビューで重要なアーキテクチャ問題を見逃す傾向。コンテキスト無視の提案も依然あり
  • ⚠️ Enterprise隠れコスト（継続） — 実質$60/人/月の「見えにくい前提」は未改善

🔑 第4回の所感

Copilot CoworkとCodex Securityでエージェント＋セキュリティの2軸を投入。 GitHub/VS Code/JetBrainsのエコシステム統合力は依然として最強。 ただし「使われているが好かれていない（42%シェア / 9%愛され率）」問題は根深い。 $10/月の手頃さは魅力だが、AI品質でClaude Code / Cursorに差をつけられている。

Windsurf

Cognition (元Codeium) — 第4回アップデート
無料プランあり Devin統合 ⚠️ 方向性変化

📢 最新アップデート

  • Devin統合プラットフォーム進展 — IDE（Windsurf）+ 自律エージェント（Devin）の統合が具体化。Linear連携の安定化が進む
  • APIセッション管理強化 — エンタープライズ向けのセッション管理・分析機能が向上
  • 永続メモリ改善 — セッション跨ぎでコードベース理解を保持する精度が向上

💰 価格体系（変更なし）

  • Windsurf Free: 基本機能（制限付き）
  • Windsurf Pro $15/月: フル機能 — 5ツール中2番目に安い
  • Devin: エンタープライズ向け（要問い合わせ）

😤 ユーザーの不満・課題（継続フォロー）

  • ⚠️ 将来の不透明さ（継続・最大の課題） — Cognition買収後のロードマップが依然として不明確。「今のWindsurfがいつまで使えるか」の不安は解消されていない
  • ⚠️ 既存ユーザーの離脱（継続） — CursorやAntigravityへの乗り換え報告が引き続きあり
  • ⚠️ 個人開発者の軽視リスク（継続） — エンタープライズ優先の姿勢が色濃い。$15/月の価格は魅力だが、機能開発がエンタープライズに偏る懸念
  • ✅ $15/月の手頃さ — コスト面では2番目に安く、価格重視ユーザーには引き続き選択肢

🔑 第4回の所感

IDE+自律エージェントの統合プラットフォーム構想は理論上最強。 しかし「理論は良いが実態が見えない」状況が長引いている。 $15/月の手頃さは引き続き魅力だが、 将来の不透明さから長期的なプロジェクトには推奨しづらい。
