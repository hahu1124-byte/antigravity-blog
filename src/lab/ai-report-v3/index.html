🔥 GPT-5.4 — AIがPCを操作する時代

3月5日リリースのGPT-5.4は、OpenAI初の「ネイティブPC操作」対応汎用モデル。 マウスクリック、キーボード入力、スクリーンショット認識をPlaywrightと連携して自律実行。 100万トークンのコンテキストで大規模コードベース全体を一度に理解し、ハルシネーション33%削減。

GitHub Copilotに即日統合され、Jiraチケットから直接PRを作成するエンタープライズワークフローが現実になりました。 ただしPC操作の精度はまだ完全ではなく、セキュリティリスクやコスト課題も。

⚡ Cursor Automations — 24時間眠らないAI

CursorのAutomationsは「AIエージェントが常時オンで自律稼働する」パラダイムシフト。 Slack、GitHub PR、PagerDutyアラートからのイベントトリガーで自動起動し、 セキュリティレビュー・インシデント対応・週次ダイジェスト生成を人間なしで実行します。

ただし「常時オン」は「常時課金」。 Cursor Pro ($20/月) でも実質$60-80/月に達するヘビーユーザーが続出中。
