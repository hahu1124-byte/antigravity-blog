# Uber配達 目標設定リファレンス

記事作成時に参照する固定目標値。

## 🛡️ 最優先事項（不変）

1. **体調管理優先** — 無理はしない
2. **無事故無違反** — 安全運転が最優先

> この2点は初回記事から変わらない絶対方針。

## 📊 目標計算式

### 週間目標

| 項目 | 金額 |
|------|------|
| 正味売上目標 | ¥70,000 |
| ガソリン代見込 | ¥12,000（2026-03-12〜 高騰対応） |
| **週間目標（合計）** | **¥82,000** |

### 月間目標

```
月間目標 = ¥82,000 × (月の日数 ÷ 7)
```

| 月の日数 | 週換算 | 月間目標（ガソリン代込） |
|---------|--------|----------------------|
| 28日 | 4.000 | ¥328,000 |
| 30日 | 4.286 | ¥351,429 |
| 31日 | 4.429 | ¥363,143 |

### 体調不良時の補正

体調不良で稼働できない日があった場合、**実稼働日数ベース**で日割り計算する。

```
補正後目標 = ¥82,000 × (実稼働可能日数 ÷ 7)
```

## 📝 記事作成時の使い方

- **日報**: 当日の売上を週間目標¥82,000に対する進捗として記載
- **週報**: 週間¥82,000達成/未達成を基準に評価
- **月報**: 月間目標に対する達成率を算出
